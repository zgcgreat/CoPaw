# 业务概览页面 UI 设计规范

## 概述

业务概览页面（BusinessOverview）用于银行管理层查看 AI 平台使用情况和业务覆盖情况。本文档定义页面的视觉设计规范，确保界面简洁专业、数据清晰可读。

## 设计原则

- **简洁专业**：减少视觉噪音，突出数据本身
- **一致性**：同类组件使用统一风格
- **可读性**：数据层次分明，信息传达清晰

## 配色方案

采用蓝白简洁系为主体，柱状图增加圆角优化。

### 主色板

| 用途 | 颜色名 | 色值 | 使用场景 |
|------|--------|------|----------|
| 主色 | 蓝色 | #1890ff | 指标卡片边框、折线图、柱状图第1位 |
| 成功 | 绿色 | #52c41a | 增长指标、柱状图第2位 |
| 警告 | 黄色 | #faad14 | 指标卡片边框、柱状图第3位 |
| 强调 | 紫色 | #722ed1 | 指标卡片边框、柱状图第4位 |

| 危险 | 红色 | #f5222d | 下降指标 |

## 组件设计

### 1. 指标卡片

**布局**：4列网格，等宽分布

**样式**：
- 背景：白色 (#fff)
- 边框：1px solid #e8e8e8
- 左边框：3px 宽彩色条（区分不同指标）
- 圆角：4px
- 内边距：16px

**结构**：
```
┌─────────────────────┐
│▌标签 (12px #8c8c8c) │
│▌数值 (28px #262626) │
│▌变化 (12px +/-颜色) │
└─────────────────────┘
```

**颜色分配**：
- 总调用次数：左边框 #1890ff
- Token消耗：左边框 #52c41a
- 使用用户：左边框 #722ed1
- 接入平台：左边框 #faad14

### 2. 折线图

**样式**：
- 背景：白色
- 边框：1px solid #f0f0f0，圆角 4px
- 网格线：虚线 #f0f0f0
- 线条粗细：2px
- 无面积填充

**线条样式**：
| 数据系列 | 颜色 | 样式 |
|----------|------|------|
| 调用次数 | #1890ff | 实线 |
| Token消耗 | #52c41a | 虚线 (6,3) |
| 用户数 | #faad14 | 实线 |

**坐标轴**：
- Y轴刻度：右侧对齐，字体 10px #999
- X轴刻度：底部居中，显示日期（MM-DD格式）

### 3. 饼图（环形图）

**样式**：
- 类型：环形图（Donut）
- 环形宽度：16px
- 背景：白色
- 无特殊效果（无发光、无阴影）

**配色**：使用主色板前5色

**图例**：
- 位置：右侧垂直排列
- 样式：色块（10x10px，圆角2px）+ 文字（12px #595959）
- 格式：名称 + 百分比

### 4. 柱状图（横向）

**样式**：
- 背景：白色
- 轨道背景：#f0f0f0
- 柱条：纯色填充，**圆角 4px**
- 柱条高度：22px
- 间距：12px

**颜色序列**（按顺序）：
1. #1890ff（蓝）
2. #52c41a（绿）
3. #faad14（黄）
4. #722ed1（紫）
5. #e866a8（玫红）

**结构**：
```
[标签 80px] [████████████ 柱条] [数值 60px]
```

### 5. 用户排行榜

**布局**：列表形式，每项一行

**排名样式**：
| 排名 | 背景 | 数字颜色 |
|------|------|----------|
| 1 | 渐变 #fbbf24 → #f59e0b | 白色 |
| 2 | 渐变 #94a3b8 → #64748b | 白色 |
| 3 | 渐变 #cd7f32 → #b45309 | 白色 |
| 4+ | #e8e8e8 | #8c8c8c |

**行样式**：
- 背景：#fafafa
- 圆角：4px
- 内边距：10px 12px
- 徽章：圆形，24x24px（前三名）/ 22x22px（其他）

## 布局规范

### 整体结构

```
┌─────────────────────────────────────────────┐
│ 筛选工具栏（日期选择 + 平台筛选）            │
├─────────────────────────────────────────────┤
│ 核心运营指标（4个指标卡片）                   │
├─────────────────────────────────────────────┤
│ 趋势图卡片                                   │
├─────────────────────────────────────────────┤
│ 热门技能 + 热门MCP服务（2列）                │
├─────────────────────────────────────────────┤
│ 模型使用分布 + Token消耗（2列）              │
├─────────────────────────────────────────────┤
│ 用户分析区域                                 │
│  ├─ 会话指标卡片（2列）                      │
│  ├─ 平台分布图（2列）                        │
│  └─ 用户排行榜（2列）                        │
└─────────────────────────────────────────────┘
```

### 间距

- 区块间距：24px
- 卡片内边距：16-20px
- 网格间距（gutter）：16px
- 元素间距：12px

## 响应式适配

| 断点 | 布局调整 |
|------|----------|
| ≥992px (lg) | 4列指标卡片，2列图表并排 |
| 768-991px (md) | 2列指标卡片，图表单列堆叠 |
| <768px (xs) | 单列布局，所有组件堆叠 |
